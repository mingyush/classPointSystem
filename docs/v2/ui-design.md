# 班级积分管理系统 V2 - UI/UX设计规范

## 一、视觉设计系统

### 1.1 品牌色彩

#### 主色调

| 名称 | 色值 | 用途 |
|------|------|------|
| Primary | `#409EFF` | 主要按钮、链接、强调元素 |
| Primary Light | `#66B1FF` | 悬停状态 |
| Primary Dark | `#337ECC` | 按下状态 |
| Primary Bg | `#ECF5FF` | 背景色 |

#### 功能色

| 名称 | 色值 | 用途 |
|------|------|------|
| Success | `#67C23A` | 成功状态、加分 |
| Warning | `#E6A23C` | 警告状态 |
| Danger | `#F56C6C` | 错误状态、减分 |
| Info | `#909399` | 信息提示 |

#### 中性色

| 名称 | 色值 | 用途 |
|------|------|------|
| Text Primary | `#303133` | 主要文字 |
| Text Regular | `#606266` | 常规文字 |
| Text Secondary | `#909399` | 次要文字 |
| Text Placeholder | `#C0C4CC` | 占位文字 |
| Border Base | `#DCDFE6` | 边框 |
| Border Light | `#E4E7ED` | 浅边框 |
| Background | `#F5F7FA` | 页面背景 |
| White | `#FFFFFF` | 卡片背景 |

## 二、组件规范

### 2.1 按钮

#### 类型

| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 蓝色背景 | 主要操作 |
| Success | 绿色背景 | 确认、加分 |
| Danger | 红色背景 | 删除、减分 |
| Default | 白色边框 | 次要操作 |

| 尺寸 | 高度 | 内边距 | 字号 |

| Large | 40px | 16px 24px | 16px |

#### 输入框

- 高度: 32px (default) / 40px (large)
- 边框: 1px solid #DCDFE6
- 圆角: 4px
- 聚焦: 边框变为主题色

| 状态 | 边框色 | 说明 |

## 五、交互规范

### 5.1 加载状态

- **按钮加载**: 显示loading图标，禁用按钮
- **页面加载**: 显示骨架屏
- **列表加载**: 底部显示加载更多
- **图片加载**: 显示占位图

### 5.2 反馈提示

| 类型 | 持续时间 | 用途 |
|------|---------|------|
| Success | 2秒 | 操作成功 |
| Warning | 3秒 | 警告提示 |
| Error | 需手动关闭 | 错误提示 |
| Info | 2秒 | 一般信息 |

### 5.3 确认对话框

危险操作需要二次确认：
- 删除学生
- 积分清零
- 归档学期
- 批量操作

### 5.4 表单验证

- **实时验证**: 输入时即时反馈
- **提交验证**: 提交前完整验证
- **错误提示**: 字段下方显示具体错误

## 六、动效规范

### 6.1 过渡动画

```css
/* 基础过渡 */
--transition-fast: 150ms ease;
--transition-base: 200ms ease;
--transition-slow: 300ms ease;
```

### 6.2 动画效果

| 动画 | 时长 | 用途 |
|------|------|------|
| fadeIn | 200ms | 元素显示 |
| fadeOut | 200ms | 元素隐藏 |
| slideIn | 300ms | 侧边栏展开 |
| slideOut | 300ms | 侧边栏收起 |
| scale | 150ms | 按钮点击 |

### 6.3 减少动画

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 七、无障碍设计

### 7.1 语义化标签

- 使用正确的HTML语义标签
- 合理的标题层级 (h1-h6)
- 列表使用 ul/ol/li

### 7.2 键盘导航

- 所有交互元素可通过Tab访问
- 支持回车键触发
- 焦点状态清晰可见

### 7.3 ARIA属性

- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1
- 不仅仅依靠颜色传达信息
