# 班级积分系统 - 用户使用指南

## 系统概述

班级积分系统 是一个专为单班级设计的简洁积分管理工具，支持本地部署和云端部署。系统提供教室大屏和管理后台两个独立入口，实现积分管理的数字化和自动化。

### 主要功能
- **双模式大屏**: 平时模式显示排行榜，上课模式支持快速积分操作
- **权限分离管理**: 班主任完整权限，任课老师仅积分操作权限
- **学生自助查询**: 学号登录查看个人积分和排名
- **奖惩项快捷操作**: 预设常用奖惩项，一键完成积分操作
- **商品预约系统**: 简化的奖品兑换预约流程

### 部署方式
- **本地部署**: Node.js + SQLite，适合校内网环境
- **云端部署**: Cloudflare Workers + D1，适合多地访问

## 系统界面介绍

### 1. 教室大屏界面
访问地址: `http://localhost:3000/display`

#### 平时模式功能
- 实时显示积分排行榜（总榜、日榜、周榜）
- 底部提供学生查询入口
- 支持全屏显示和自动刷新

#### 上课模式功能
- 显示学生列表（按学号排序）
- 显示常用奖惩项按钮
- 支持快速积分操作
- 显示当前教师和上课时间

#### 操作说明
1. **模式切换**: 点击界面上的模式切换按钮
2. **学生查询**: 在平时模式下，学生可在底部输入学号查询
3. **全屏显示**: 按 `F11` 进入全屏，按 `Esc` 退出
4. **自动切换**: 上课模式2小时后自动切换回平时模式

### 2. 管理后台界面
访问地址: `http://localhost:3000/admin`

#### 登录方式
- **班主任账户**: admin / admin123 (完整权限)
- **任课老师账户**: teacher / 123456 (仅积分操作)

#### 班主任功能

**2.1 学生管理**
- 添加、编辑、删除学生信息
- 查看学生积分详情
- 管理学生账户状态

**操作步骤**:
1. 进入学生管理页面
2. 点击"添加学生"按钮
3. 填写学生姓名和学号
4. 保存学生信息

**2.2 积分管理**
- 手动调整学生积分
- 查看积分操作记录
- 执行积分清零操作

**积分清零步骤**:
1. 进入积分管理页面
2. 点击"积分清零"按钮
3. 输入确认密码
4. 确认清零操作

**2.3 奖惩项管理**
- 设置常用奖惩项目
- 配置奖惩分数和类型
- 管理奖惩项显示顺序

**设置奖惩项步骤**:
1. 进入奖惩项管理页面
2. 点击"添加奖惩项"
3. 填写名称、分数、类型
4. 设置排序和启用状态

**2.4 商品管理**
- 添加/编辑/删除奖品商品
- 设置积分价格和库存
- 管理商品上架状态

**2.5 预约管理**
- 查看学生预约订单
- 确认或取消预约
- 处理积分扣除和库存

#### 任课老师功能

**2.6 积分操作**
- 为学生加分或减分
- 使用预设奖惩项快速操作
- 查看积分操作记录

**快速积分操作步骤**:
1. 在大屏上课模式下选择学生
2. 点击相应的奖惩项按钮
3. 系统自动完成积分操作
4. 实时更新排行榜显示

### 3. 学生查询功能

#### 查询方式
- 在大屏平时模式下输入学号查询
- 无需单独的学生界面，简化操作流程

#### 查询内容

**3.1 个人积分信息**
- 当前积分余额
- 班级排名位置
- 本周积分变化

**3.2 积分记录查看**
- 个人积分操作历史
- 操作时间和原因
- 操作教师信息

**3.3 商品浏览和预约**
- 查看可兑换商品列表
- 了解商品价格和库存
- 提交预约申请

**学生查询步骤**:
1. 在大屏平时模式下找到查询入口
2. 输入个人学号
3. 查看个人积分信息
4. 浏览商品并提交预约（如需要）

## 日常使用流程

### 教师日常操作

**上课时**:
1. 打开教师管理界面
2. 切换到"上课模式"
3. 大屏显示积分操作界面
4. 根据学生表现进行加减分

**平时**:
1. 切换到"平时模式"
2. 大屏显示积分排行榜
3. 处理学生商品预约
4. 管理商品库存

### 学生日常操作

**查看积分**:
1. 使用学号登录学生界面
2. 查看个人积分和排名
3. 了解积分变化情况

**预约商品**:
1. 浏览商品列表
2. 选择心仪商品
3. 确认预约
4. 联系老师完成兑换

## 常见问题解答

### Q1: 忘记教师密码怎么办？
**A**: 联系系统管理员重置密码，或查看系统配置文件。

### Q2: 学生积分显示不正确怎么办？
**A**: 
1. 检查网络连接
2. 刷新页面
3. 联系教师核实积分记录

### Q3: 大屏显示不更新怎么办？
**A**: 
1. 检查网络连接
2. 按F5手动刷新
3. 重新打开页面

### Q4: 商品预约后如何取消？
**A**: 
1. 联系教师
2. 教师在管理界面取消预约
3. 积分自动返还

### Q5: 系统运行缓慢怎么办？
**A**: 
1. 关闭不必要的浏览器标签页
2. 清理浏览器缓存
3. 重启系统

## 系统维护

### 日常维护

**数据备份**:
- 系统自动备份数据文件
- 建议定期手动备份重要数据
- 备份文件位于 `data/backups` 目录

**性能监控**:
- 定期检查系统性能
- 关注内存和CPU使用情况
- 及时清理过期数据

### 故障处理

**系统无法启动**:
1. 检查Node.js是否正常安装
2. 确认端口3000未被占用
3. 查看错误日志文件

**数据丢失**:
1. 从备份文件恢复数据
2. 检查数据文件权限
3. 联系技术支持

**网络连接问题**:
1. 检查网络设置
2. 确认防火墙配置
3. 重启网络服务
